## 目录结构

```
|- build/
|- apps/ 
|- resetPwd/  --------------------------------- 项目代码
  |- components/  ----------------------------- 组件目录
    |- App.vue
    |- Step.vue  ------------------------------ 步骤条组件
    |- First.vue  ----------------------------- 第一页
    |- ...
  |- router/  --------------------------------- 路由文件配置
  |- app.js  ---------------------------------- webpack入口文件
|- views/
  |- resetPwd.html  ------------------------------ HTML 模板
|- .babelrc  ---------------------------------- babel 配置文件
|- package.json  ------------------------------ npm 配置文件
|- server.js  --------------------------------- 项目入口文件
|- webpack.config.js  ------------------------- webpack 配置文件
```
